Purpose of 'package' (line 1)? - To group together code with a similar purpose
Why do we use 'import' statements? - To give our package access to code written in another package

How to run code? - 'go run helloworld/main.go'

Commands:
	'go build' - Compiles a bunch of go source code files
	'fo fmt' - Formats all the code in each file in the current directory
	'go install' - Compiles and 'installs' a package
	'go get' - Downloads the raw source code of someone else's package
	'go test' - Runs any tests associated with the current project
